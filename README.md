# TD5 - Application Django de Taux de Change BCM

Application web Django pour visualiser les taux de change MRU vers USD, EUR et CNY.

## Fonctionnalités
- Affichage des taux de change
- Mise à jour manuelle des taux
- Stockage SQLite

## Technologies
- Django 5.2.9
- Python 3.12
- Docker & Docker Compose
